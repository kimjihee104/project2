github.

1. github에서 repository 를 만든다

2. 로컬 밑에 폴더를 만들어준다 (git-project)

3. git에서 만든 것 내 폴더 아래로 저장하기
<cmd 창> 
>cd git-project
>git clone https://github.com/kimjihee104/project1.git

(그럼 git-project에 project1 파일이 생성한다)

4. 내용 추가 할 경우
>cd project1
>git add .
>git commit -m''
>git push
